# Positions

Widgets for displaying position data and trade activity. All require `FunctionSpaceProvider`.

| Widget | Description |
|--------|-------------|
| **PositionTable** | Paginated table with Open/Closed/History tabs, row selection, sell actions, and P/L display |
| **TimeSales** | Live trade feed showing recent buy/sell activity with optional polling |
