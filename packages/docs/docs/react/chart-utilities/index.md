# Chart Utilities

Hooks and helpers for implementing interactive Recharts-based charts with zoom and pan.

| Export | Type | Description |
|--------|------|-------------|
| **useChartZoom** | Hook | Manages zoom/pan state for Recharts charts with mouse wheel, drag, and double-click reset |
| **rechartsPlotArea** | Factory function | Computes plot area boundaries from a Recharts container ref and margin config |
