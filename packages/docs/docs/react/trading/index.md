# Trading - React

State management hook for custom shape editing. Requires `FunctionSpaceProvider`.

| Hook | Description |
|------|-------------|
| **useCustomShape** | Manages control point values, locks, and drag state for the interactive shape editor. Derives a normalized `BeliefVector` from current control values. |
