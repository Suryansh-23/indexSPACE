# Auth - React

React hook for accessing authentication state and actions from `FunctionSpaceProvider`.

| Hook | Description |
|------|-------------|
| **useAuth** | Accessor hook exposing `user`, `isAuthenticated`, `login()`, `signup()`, `logout()`, and related auth state from context |
